PYP's Tour

Proposed by: Param, Yesha, Preet |

Description

Our idea is to design a web application for tours and travels to manage tours, packages, and customer bookings. This system allows customers to search available tours, view detailed information, and submit booking requests through the website. Admin can manage tour listings, update information, and monitor customer bookings through a structured management interface.

This application needs an admin system so that tours and packages can be added, deleted, and updated.

Customers – Public Users

  • View Tour Listings
  • View Detailed Tour Pages
  • Submit Booking Requests
  • Send Contact Enquiries
  • Navigate Through the Website

Dashboard for Admin

  • Add New Tours
  • Edit Existing Tours
  • Delete Tours
  • View Customer Bookings
  • Manage Enquiries

User Requirements – User Stories

  • As a visitor, I want to browse available tours so that I can choose a destination.
  • As a visitor, I want to view detailed information about a tour so that I can understand the package.
  • As a visitor, I want to submit a booking request so that I can reserve my trip.
  • As a visitor, I want to send an enquiry so that I can ask questions.

Admin User Stories

  • As an admin, I want to add a new tour so that new travel packages can be offered.
  • As an admin, I want to update tour details so that information stays accurate.
  • As an admin, I want to delete tours so that outdated packages are removed.
  • As an admin, I want to view bookings so that I can manage customer reservations.
