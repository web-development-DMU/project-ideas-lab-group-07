Ticket Booking System

Proposed by: P2877257 & P2835206

Description

This is ticket booking app where you can buys tickets for football matches. The ticket buyers can buy tickets and can cancel at their convenience. It needs an admin system so administrators can set up staff and module records and create programmes as collections of modules.
