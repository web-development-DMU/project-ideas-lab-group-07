Library Management System

Library Management System

Proposed by: Bhavya Praisy Said

Description

The Library Management System is a web-based application designed to manage books, members, and borrowing records efficiently. It allows librarians to perform CRUD operations, track issued and returned books, and maintain organized records. This project demonstrates practical skills in database management and full-stack web development while providing a user-friendly solution to streamline library operations.

core features

  • Book Management
  • Member Management
  • Borrowing System
  • Admin Dashboard
