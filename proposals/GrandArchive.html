Library Management System

Proposed by: Bhavya Praisy Said

Description

The Library Management System is a web-based application designed to efficiently manage library resources, including books, members, and borrowing records. The system aims to streamline the process of cataloguing books, tracking loans, and managing user accounts, providing a structured and user-friendly solution for both librarians and members. This project demonstrates practical implementation of database management, CRUD operations, and full-stack web development principle.

core features

  • Book Management
  • Member Management
  • Borrowing System
  • Admin Dashboard
