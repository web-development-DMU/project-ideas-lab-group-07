Library management

Proposed by: BHAVYA, PRAISY, SAID

Description

The Library Management System is a web-based application designedto manage books, members, and borrowing records efficiently. Traditional library systems often lack digital coordination and personalised search systems. This project addresses such issues by introducing a digital platform that filters books by course, supports online reservation with physical collections and facilitates online catalogue searches to identify and verify the availability of the physical books within the library. It also allows librarians to perform CRUD operations, track issued and returned books, and maintain organized records. This project demonstrates practical skills in database management and full-stack web development while providing a user-friendly solution to streamline library operations.

core features

  • Book Management
  • Member Management
  • Borrowing System
  • Admin Dashboard
  • Create/edit/delete members profile
  • Create/edit/delete modules
